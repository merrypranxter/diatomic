# Contributing

This project is an art–science sandbox for diatom shapes, structure, and math.

## Ways to contribute

- Add a new **shape family** page under `catalog/centric` or `catalog/pennate`.
- Add a **species profile** with images, outlines, and references.
- Add an **outline dataset** under `data/outlines` and corresponding descriptors.
- Add a **notebook** or **shader** that analyzes or generates diatom-like forms.

## Style & conventions

- Use descriptive filenames, e.g. `disc_radiate_valve_view.md`, `naviculoid_boat_shapes.md`.
- In catalog entries, include:
  - A short visual description.
  - Notes on structure and anatomy.
  - Links to data files, images, and any relevant code.
  - References to the literature or keys where similar forms are described.
- Document image sources and licenses clearly.

## Process

1. Fork the repo.
2. Create or edit files in a logically small change set.
3. Include references where you make biological or mathematical claims.
4. Open a pull request with a brief description and, if relevant, example images.
