# Pennate catalog

Pennate diatoms typically have elongated, bilaterally symmetric valves with ornamentation organized along a long axis, often with a raphe system.

Shape families here include naviculoid boats, cymbelloid bananas, sigmoids, ribbon-forming forms, and more.
