# Catalog

This directory is the visual atlas portion of the project.

- [`centric/`](centric/README.md) – shape families with radial or rotational symmetry in valve view.
- [`pennate/`](pennate/README.md) – shape families with bilateral or related symmetries.

Each page describes:

- A **shape family** (outline + symmetry).
- The **view** (valve or girdle).
- Structural notes and example taxa.
- Pointers to images, outline data, and code.

This is intentionally not a formal identification key, but it should be anchored enough in real morphology that phycologists can recognize what is being referenced.
