# Centric catalog

Centric diatoms typically have radial or rotational symmetry in valve view, with silica ribs and striae radiating from a centre or annulus.

Here, shape families are defined visually (disc, polygon, stellate, cylindrical chain formers) rather than by formal taxonomy.
